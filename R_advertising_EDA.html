c.Mode

R - does not have an inbuilt R code for mode. Hence we will create our own function.

getmode <- function(v){
  uniqv <- unique(v)
  uniqv[which.max(tabulate(match(v ,uniqv)))]
}
#i. Find the mode age
mode_age <- getmode(age)
mode_age
